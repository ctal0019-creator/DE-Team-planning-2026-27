GitHub Pages upload pack for the Digital Education Planning app.

Files in this folder:
- `index.html` - planning app
- `dashboard.html` - dashboard
- supporting CSS and JS files
- `.nojekyll` - helps GitHub Pages serve static files cleanly

Quick publish steps:
1. Create a new GitHub repository.
2. Upload everything from this folder to the top level of that repository.
3. In GitHub, open `Settings` > `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Choose branch `main` and folder `/ (root)`.
6. Save and wait for the GitHub Pages link to appear.

Important:
- The planning app currently saves data only in each person's own browser.
- That means GitHub Pages will host the app, but it will not create a shared live team database.
- People can still use it, download their workbook, and send it on for combining.
